# Lab4_Productos — Directorio de Productos ASP.NET Core MVC

Aplicación web desarrollada en **ASP.NET Core MVC (.NET 10)** para la gestión de un catálogo de productos en memoria.

---

## ⚠️ Instrucciones para correr el proyecto !!!!!

Por seguridad, se recomienda correr la aplicación **desde la terminal** con el siguiente comando:

### 1. Abre una terminal y navega a la carpeta del proyecto

```bash
cd "Aplicacion Web Productos"
```

### 2. Ejecuta la aplicación

```bash
dotnet run
```

### 3. Abre el navegador en la URL que aparece en la terminal
